# redding-open-budget-web Open-source civic project making the City of Redding's budget readable for everyone. [README.md](https://github.com/user-attachments/files/25789381/README.md) # Redding Open Budget Redding Open Budget is an open-source civic project that consolidates and organizes the City of Redding's public expenditures in a single, accessible place. The goal is to offer a clear, structured, and auditable view of how public money was actually used — bringing together data today scattered across budget PDFs, financial reports, and government portals. More than a visualization, this project proposes a consolidated model for reading municipal spending. --- ## 🎯 Purpose Redding Open Budget aims to: - Consolidate City of Redding spending data into a unified, navigable interface - Display only amounts **actually paid**, not just budgeted or appropriated - Exclude noise such as enterprise fund pass-throughs and debt refinancing rollovers - Reclassify expenditures to allow better understanding of where money truly goes - Make public data accessible without sacrificing methodological rigor The focus is on balancing clarity for general residents and consistency for technical users, journalists, and researchers. --- ## 🏛 Current Scope The project currently covers: - City of Redding **General Fund** — FY 2024-25 - Spending by department: Police, Fire, Public Works, Community Services, Administration, Finance & Legal Data is structured hierarchically: **Department → Division → Expenditure Category** --- ## 📊 Methodology The project considers only amounts **effectively paid** during the fiscal year. Excluded: - Enterprise funds (Redding Electric Utility, water, wastewater, airport) — self-funded by ratepayers, not taxpayers - Pass-through funds and state/federal grants not under city discretion - Debt refinancing rollovers — only interest payments (debt service) are counted - Budget appropriations not yet disbursed The full methodology is documented on the site at `/methodology`. --- ## 🗂 Data Sources All data is obtained exclusively from official sources: - [City of Redding Biennial Budget FY 2023-24 & 2024-25](https://www.cityofredding.gov/government/departments/finance/budget) - [California State Controller — Government Financial Reports](https://www.sco.ca.gov) - [Transparent California — Employee Compensation](https://transparentcalifornia.com) - [Open FI$Cal — California Expenditure Data](https://open.fiscal.ca.gov) All datasets are public and independently verifiable. --- ## 🛠 Tech Stack **Frontend** - React + Vite + TypeScript - Tailwind CSS - Recharts **Data Pipeline** *(Phase 2)* - Python (CSV processing from Open FI$Cal) - Supabase (database + API) **Infrastructure** - GitHub (version control) - Vercel (hosting, free tier) --- ## 🚀 Roadmap **Phase 1 — MVP (current)** - [x] Static site with FY 2024-25 General Fund data - [x] Home summary with spending breakdown - [x] Department drill-down - [x] Methodology, Data Sources, and About pages - [ ] GitHub repo + Vercel deploy **Phase 2 — Real data pipeline** - [ ] Python processor for Open FI$Cal CSV files - [ ] Supabase integration - [ ] FY selector (multiple years) - [ ] Department search **Phase 3 — Analysis layer** - [ ] Editorial analysis pages (Public Safety, Debt Service, Pensions, Homelessness) - [ ] Public API with documentation - [ ] Redding vs. comparable cities comparison --- ## 📌 Project Principles - Methodological transparency - Political neutrality - Technical rigor - Open source - Verifiable data --- ## 🤝 How to Contribute This project is open to technical and non-technical contributions.
